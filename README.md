# GoNext — Дневник туриста

Мобильное приложение на **Expo Router** + **TypeScript** с **React Native Paper** (Expo SDK 54).

## Основные экраны

- **Главный** — навигация: Места, Поездки, Следующее место, Настройки
- **Места** — список, добавление, карточка с фото, открытие на карте
- **Поездки** — список, создание, маршрут с местами, отметки посещений
- **Следующее место** — первое непосещённое место текущей поездки, кнопки карта/навигатор
- **Настройки** — о приложении, сброс данных

## Офлайн

Приложение работает без интернета. Данные хранятся локально (SQLite, файловая система). Сеть используется только при нажатии «Открыть на карте» или «Открыть в навигаторе».

## Запуск

```bash
npm install
npx expo start
```

- **iOS/Android:** отсканируйте QR-код в Expo Go
- **Web:** в меню нажмите `w` или `npx expo start --web`

Проверка совместимости: `npx expo-doctor`
